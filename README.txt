Инструкция по установке и запуску скрипта парсинга Росеестра:

! Чтобы развернуть скрипт, вам желательно иметь Debian не ниже 9 версии и Ubuntu не ниже 18(можно и другие системы, но не протестировано)

1.
    Устанавливаем необходимые пакеты в системном пакетном менеджере:

    Debian:
        apt-get install firefox-esr python3-pip python3

    Ubuntu:
        apt-get install firefox python3-pip python3

    ! убедитесь что ваша версия Python не ниже 3.6, иначе скрипт работать не будет


2.
    Переходим в рабочий каталог скрипта, куда мы его развернули, и выполняем команду:

        pip3 install -r req.txt

3.
    Из каталога bin в текущем рабочем каталоге скрипта копируем исполнительный файл для управления firefox:

        cp ./bin/geckodriver /usr/local/bin/geckodriver;chown +x /usr/local/bin/geckodriver


4. Все настройки завершены, можно запускать скрипт. Всю дополнительную информацию по параметрам запуска скрипта
   можно получить запустив скрипт с ключом --help


Удачного использования!
